# **App Name**: CollectPro

## Core Features:

- Dashboard Stats: Display daily statistics like calls made, payment promises, collections, and call success ratio.
- Auto-Dialer Integration: Implement an auto-dialer feature, providing agents with a click-to-call option.
- Debtor Profile Retrieval: AI-powered tool to deliver contextually relevant data about a debtor from available databases, enhancing agent readiness.
- Call Recording & Logging: Automatically record calls and maintain a call log, adhering to compliance requirements.
- Task Management: Agents can create, view, and manage follow-up tasks with a calendar interface and reminder system.
- Secure Messaging: Facilitate internal communication through secure messaging, including SMS/email templates for debtor correspondence.
- Performance Overview: Real-time display of agent performance metrics such as call volume, collections, and success rates.

## Style Guidelines:

- Primary color: (#0f2557) to convey reliability.
- Background color:  (#3b5ba5) to maintain a clean, uncluttered layout.
- Accent color: (#e87a5d) to highlight positive actions.
- Body and headline font: 'PT Sans', a sans-serif font that is modern, neutral, and suitable for both headlines and body text.
- Crisp, professional icons that align with the serious and reliable nature of financial interactions.
- Intuitive and efficient design, focusing on usability with easy-to-navigate sidebars and top navigation.
- Subtle transitions for data loading and updates to enhance user experience without distraction.
